# Breadth First Traversal

## Problem Domain

Given a tree, write a function that returns a list of all the values in the tree, in the order they are encountered, when using a breadth-first traversal.

---

### Big O

| Time | Space |
| :----------- | :----------- |
| O(n) | O(n) |
